# 打标签网站

### 1. 安装需求

```bash
# 安装apache2
sudo apt install -y apache2
# 安装php
sudo apt install -y php
# 安装php插件
sudo apt install -y php-mysql php-xml php-mbstring php-gd
```

### 2. 安装帮助

- **创建数据库`Material`**

- **手动执行`./php/install.sql`脚本**

- **修改`./php/mysql.json`配置数据库相关信息**

- **将待处理的全部图片放在`./data/src/`路径下**

### 3. 开始使用

- 打开对应的网页开始使用
